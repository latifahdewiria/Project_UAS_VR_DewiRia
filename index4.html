Panduan Penggunaan Virtual Tour

1. Klik tombol "Start" untuk memulai tur virtual.

2. Klik tombol "VR" di bagian bawah sebelah kanan untuk melihat virtual tour secara full screen.

3. Gunakan kursor lingkaran berwarna hitam untuk menavigasi dan klik pada objek yang diinginkan untuk melihat detail lebih lanjut.

4. Gunakan tombol kembali di sudut kiri atas untuk kembali ke halaman utama.

Tentang Virtual Tour

Virtual tour ini dibuat untuk menampilkan berbagai sekolah dan perguruan tinggi di Belitang. Anda dapat menavigasi melalui gambar-gambar 360 derajat dan mendapatkan informasi tentang masing-masing institusi.
